# Security Policy

Please do not include private transcripts, secrets, access tokens, customer data, or sensitive incident details in public issues.

For security-sensitive reports, contact the maintainer privately before publishing details. Include the affected version or commit, the command you ran, and the smallest safe reproduction you can share.

`agent-run-audit-skill` reads local transcript files and writes local audit artifacts. Report any behavior that unexpectedly executes transcript commands, contacts external services, or exposes private transcript content.
